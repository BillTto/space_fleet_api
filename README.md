# Space Fleet API

Учебный REST API на Flask для управления космическим флотом.

Проект реализует работу с космическими кораблями и миссиями,
включая создание ресурсов, обновление статусов, добавление кораблей в миссии
и обработку ошибок.

## Возможности
- CRUD для космических кораблей (spaceships)
- Создание и просмотр миссий (missions)
- Добавление корабля в миссию
- Корректные HTTP-статусы (200, 201, 400, 404)
